# Flood Mapping of Coastal Cuddalore Web GIS Dashboard

## Project Overview
This Web GIS Dashboard presents the coastal flood susceptibility mapping of Cuddalore District, Tamil Nadu.  
The dashboard was developed to visualize important vector and raster layers related to coastal flood analysis in an interactive environment.

## Project Overview
This Web GIS Dashboard presents the coastal flood susceptibility mapping of Cuddalore District, Tamil Nadu.  
The dashboard was developed to visualize important vector and raster layers related to coastal flood analysis in an interactive environment.

## Live Dashboard Link
[Open Dashboard](https://prasanthvvprasanth7-art.github.io/cuddalore-webgis-dashboard/)

## Study Area
Cuddalore District, Tamil Nadu, India

## Theme
Coastal Flood Susceptibility Mapping

## Methodology
This project integrates:
- GIS-based thematic layer preparation
- Remote sensing data processing
- Machine Learning approach (Random Forest)
- Interactive Web GIS visualization

## Data Layers Included

### Vector Layers
- Boundary
- Distance from Coast
- Lithology
- Roadways
- Railways
- EPR
- Soil
- Flood Zones

### Raster Layers
- Elevation Raster
- Slope Raster
- Rainfall Raster
- BD Raster
- DD Raster
- LULC Raster
- SSZ Raster
- TIZ Raster

## Dashboard Features
- Interactive Web GIS map
- Layer on/off control
- Raster and vector layer separation
- Download option for layers
- Basemap switch
- Project presentation section

## How to Use
1. Open the dashboard link
2. Use the checkboxes in the sidebar to turn layers on or off
3. Use the basemap switch to change between OpenStreetMap and Satellite view
4. Use the layer download icons to download available data
5. Zoom and pan the map to explore the study area
